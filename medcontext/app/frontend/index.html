Multi-modal Document Ingestion

Extracts structured data from handwritten clinical notes, PDFs, radiology reports, lab results, and audio transcripts using Docling and LightOnOCR-2-1B.

Docling LightOnOCR Qwen3-VL Whisper
🤖

Multi-agent Orchestration

Specialized AI agents collaborate via LangGraph — a retrieval agent, a drug interaction agent, a diagnostic reasoning agent — each challenging and refining the others.

LangGraph Groq Gemini Ollama
📚

Multimodal RAG Pipeline

Hybrid retrieval using dense embeddings from Qwen3-VL-Embedding-2B and BM25 sparse search, combined via Reciprocal Rank Fusion for precision retrieval.

Qdrant RRF Qwen3-VL BM25
🧪

LLM Evals + Feedback

Every agent output is evaluated by a judge LLM. Low-confidence answers trigger deeper reasoning loops. Results feed back to improve retrieval quality.

LLM-as-judge LangFuse Feedback loop
🔒

Safety & Privacy First

Fully local deployment. Encrypted SQLite via SQLCipher. No real patient data, no cloud inference required. Rate-limit fallback: Groq → Gemini → Ollama.

SQLCipher Local-only FastAPI
📈

Knowledge Graph Integration

Drug interactions, ICD-10 codes, and clinical relationships are traversed via a medical knowledge graph to surface hidden risks and contraindications.

Graph RAG ICD-10 Drug DB
